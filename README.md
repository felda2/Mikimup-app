# Aplikasi Java GUI

Repository ini digunakan untuk mengembangkan aplikasi Java berbasis GUI.

## Teknologi
- Java
- Java Swing
- MySQL
- GitHub
- Visual Studio Code

## Cara Menjalankan Program
1. Pastikan Java JDK sudah terinstal.
2. Buka proyek menggunakan Visual Studio Code.
3. Buka file `src/main/Main.java`.
4. Klik tombol **Run**.
5. Login menggunakan:
   - Username: `admin`
   - Password: `123`

## Struktur Folder
- `model`: Menyimpan class data.
- `service`: Menyimpan logika aplikasi.
- `view`: Menyimpan halaman GUI.
- `database`: Menyimpan konfigurasi database.
- `util`: Menyimpan fungsi tambahan.
